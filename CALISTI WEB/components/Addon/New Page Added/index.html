OPENClarity

OpenClarity is a suite of open source tools for cloud-native security and observability—APIClarity, KubeClarity, and FunctionClarity. These open source technologies are the foundation for Cisco’s cloud-native application security platform, Panoptica.

FunctionClarity provides a framework to sign and validate cloud function core artifacts (images or code) with various signing techniques including keyless (utilizing the cosign/sigstore project).
